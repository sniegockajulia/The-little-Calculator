The Little Calculator

  • I => Enter input field
  • E => Enter exponent field
  • + => Add number
  • * => Multiply number
  • = => Equals
  • Esc => Clear
  • ^ (Shift + 6) => Exponentiate number
  • # (Shift + 3) => Square root of a number
  • | (Shift + \) => Modulo of a number
  • ! (Shift + 1) => Factorial of a number
  • L => Download error log
  • S => Sum the CSV list elements
  • O => Sort (order) the CSV list elements
  • R => Reverse the CSV list elements
  • D => Delete an element from the CSV list
  • A => Average of the CSV list elements
  • Shift + D => Enter “delete element” field
  • Shift + Y => Confirm deleting element
  • Shift + N => Cancel deleting element
